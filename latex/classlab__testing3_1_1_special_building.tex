Рассчитывает стоимость здания с учетом типа. 

Расчет стоимости производится c разным коэффицентом взависимости от типа специального здания
\begin{DoxyItemize}
\item Формула расчета стоимости:\+   \[    C = S \times P \times k(t)
\] где:\+
\item $ C $ — общая стоимость здания
\item $ S $ — площадь здания (м²)
\item $ P $ — стоимость за м²
\item $ k(t) $ — коэффициент типа здания:\+
\begin{DoxyItemize}
\item $ k(0) = 0.3 $
\item $ k(1) = 1.0 $
\item $ k(2) = 0.5 $
\end{DoxyItemize}
\end{DoxyItemize}
